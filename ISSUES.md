# lamos Issue Guide

Please keep issues short.

All we need:
- A short but detailed description of your issue (more you provide then better)
- When it happened - what you were doing (analyzing image, files etc.)
- Screenshots (assistant/problem/settings)

Include these details:
- App version, device, iOS version, model used
- Steps to reproduce
